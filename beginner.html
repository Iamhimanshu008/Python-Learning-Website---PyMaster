üì¶ What are Variables?

A variable is a container where we store data. Like a box where we keep things!

üí°
Analogy: Variable = Labeled Box. Just like a box labeled "Sugar" stores sugar, name = "Himanshu" stores "Himanshu" inside name!

Loops ‚Äî for & while üîÑ

‚ñº

üîÑ What are Loops?

Loops allow us to do a task repeatedly ‚Äî without writing code again and again!

üí°
Analogy: Imagine you have to write "I love Python" 100 times. Will you write print() 100 times? No! Use a Loop ‚Äî done in one line!

üîÅ for Loop

Python 
# Example 1: With range()
for i in range(5):
    print(f"Count: {i}")
# 0, 1, 2, 3, 4

# Example 2: Iterate a list
fruits = ["Apple", "Banana", "Mango", "Orange"]
for fruit in fruits:
    print(f"I like {fruit}! üçé")

# Example 3: range(start, stop, step)
for i in range(2, 11, 2):
    print(i, end=" ")  # 2 4 6 8 10

# Example 4: Iterate a string
for char in "Python":
    print(char, end="-")  # P-y-t-h-o-n-

üîÑ while Loop

Python 
# Example 1: Simple countdown
count = 5
while count > 0:
    print(f"‚è≥ {count}...")
    count -= 1
print("üöÄ Launch!")

# Example 2: User input loop
while True:
    answer = input("Quit? (yes/no): ")
    if answer.lower() == "yes":
        print("Bye! üëã")
        break

# Example 3: Sum of numbers
total = 0
num = 1
while num <= 10:
    total += num
    num += 1
print(f"Sum of 1-10: {total}")  # 55

‚ö° break, continue, pass

Python 
# break ‚Äî Exit the loop
for i in range(10):
    if i == 5:
        break   # Stop at 5
    print(i)    # 0 1 2 3 4

# continue ‚Äî Skip current iteration
for i in range(6):
    if i == 3:
        continue  # Skip 3
    print(i)      # 0 1 2 4 5

# pass ‚Äî Do nothing (placeholder)
for i in range(3):
    pass  # Write code later

print(msg.split())        # Break into List

print(msg.count("l"))     # 2 ‚Äî how many times appeared
print(msg.find("Python")) # 8 ‚Äî where found
print(msg.startswith(" ")) # True
print("-".join(["A","B","C"]))  # A-B-C
