‚ùì Top 25 Python Interview Questions

Interviews mein puchhe jaane wale important questions with answers!

Q1. Python mein List aur Tuple mein kya difference hai?

List: Mutable (change ho sakti), [] se banate hain. Tuple: Immutable (change nahi hoti), () se banate hain. Tuple faster hai aur dictionary key ban sakti hai.

Q2. Python mein == aur is mein kya fark hai?

== value compare karta hai, is identity (memory location) compare karta hai. a = [1,2]; b = [1,2] ‚Üí a == b is True, a is b is False.

Q3. *args aur **kwargs kya hain?

*args = Multiple positional arguments (tuple mein aate hain). **kwargs = Multiple keyword arguments (dictionary mein aate hain).

Q4. Python mein Decorator kya hota hai?

Decorator ek function hai jo dusre function ko modify karta hai bina uska code change kiye. @decorator_name syntax se use hota hai. Example: logging, timing, authentication.

Q5. Python mein GIL (Global Interpreter Lock) kya hai?

GIL ek mutex hai jo CPython mein ek time pe sirf ek thread ko Python bytecode execute karne deta hai. Isliye truly parallel CPU-bound tasks ke liye multiprocessing use karna padta hai.

Q6. List Comprehension kya hai?

List banana ka short way: [expression for item in iterable if condition]. Example: [x**2 for x in range(10) if x%2==0] = squares of even numbers.

Q7. Python mein shallow copy aur deep copy mein kya fark hai?

Shallow copy: New object banata hai but nested objects share hote hain. Deep copy: Completely independent copy banata hai. import copy; copy.deepcopy(obj)

Q8. self kya hai Python classes mein?

self current object ka reference hai. Har method ka first parameter hota hai. Isse hum object ke attributes aur methods access karte hain.

Q9. Python mein Generator kya hai?

Generator ek function hai jo yield keyword use karta hai (return ki jagah). Ye lazy evaluation karta hai ‚Äî values ek-ek karke generate hoti hain, sab ek saath memory mein load nahi hoti.

Q10. Python mein __init__ method kya karta hai?

__init__ constructor hai ‚Äî jab object create hota hai tab automatically call hota hai. Isme hum object ke initial attributes set karte hain.

Q11. Python mein mutable aur immutable types kya hain?

Mutable: list, dict, set (change ho sakte). Immutable: int, float, str, tuple, frozenset (change nahi ho sakte).

Q12. map(), filter(), reduce() kya hain?

map(fn, list) = har element pe function lagao. filter(fn, list) = condition match karne wale elements nikalo. reduce(fn, list) = sab elements ko combine karo into single value.

Q13. Python mein with statement kya hai?

with context manager hai ‚Äî resources (files, connections) ko automatically clean up karta hai. with open("file") as f: ‚Äî file automatically close ho jaati hai.

Q14. __name__ == "__main__" ka matlab kya hai?

Check karta hai ki file directly run ho rahi hai ya import ho rahi hai. Direct run pe __name__ = "__main__". Import pe module ka naam hota hai.

Q15. Python mein exception handling kaise karte hain?

try = code jo error de sakta hai. except = error handle karo. else = no error toh ye chale. finally = hamesha chale (cleanup). raise = apni error throw karo.

Q16. pass, continue, break mein kya fark hai?

pass = kuch nahi karo (placeholder). continue = current iteration skip karo, next pe jao. break = poora loop se bahar niklo.

Q17. Python mein inheritance kya hai?

Child class parent class ke properties aur methods inherit karti hai. class Dog(Animal): ‚Äî Dog ko Animal ke sab methods mil jaate hain. Code reuse ka best tarika.

Q18. staticmethod aur classmethod mein kya fark hai?

@staticmethod ‚Äî class/instance se independent, no self/cls parameter. @classmethod ‚Äî cls parameter leta hai, class ko modify kar sakta hai.

Q19. Python mein virtual environment kya hota hai?

Isolated Python environment ‚Äî har project ke apne alag packages hote hain. python -m venv myenv se banate hain. Dependency conflicts avoid hote hain.

Q20. pip kya hai?

pip = Python package manager. pip install package_name se packages install hote hain. pip freeze > requirements.txt se dependencies save hoti hain.

Q21. Python 2 vs Python 3 main differences?

Print: print "hi" (Py2) vs print("hi") (Py3). Division: 5/2=2 (Py2) vs 5/2=2.5 (Py3). Python 2 officially dead since 2020, always use Python 3.

Q22. zip() function kya karta hai?

Multiple iterables ko parallel mein combine karta hai. zip([1,2,3], ['a','b','c']) ‚Üí [(1,'a'), (2,'b'), (3,'c')]

Q23. Python mein enumerate() kya hai?

Loop mein index aur value dono deta hai. for i, val in enumerate(["a","b","c"]): ‚Üí i=0,val="a" etc.

Q24. Abstract class kya hoti hai?

Abstract class ka object nahi ban sakta. from abc import ABC, abstractmethod. Child class ko abstract methods implement karne padte hain. Blueprint/contract jaisi hai.

Q25. Python mein memory management kaise hoti hai?

Python automatic garbage collection use karta hai (reference counting + cyclic garbage collector). Jab kisi object ka koi reference nahi raha, memory free ho jaati hai.
